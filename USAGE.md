# ADB Cleaner v2.0 使用说明

## 快速开始

### Windows 用户

双击运行 quick-clean.bat 即可自动清理。

### OpenClaw 用户

在 OpenClaw 中说：

`
安装这个 skill：https://github.com/ihpwhath/adb-cleaner
`

然后说：

`
帮我清理手机
`

## 前置要求

1. 开启 USB 调试：设置 → 开发者选项 → USB 调试
2. 获取 Root 权限：KernelSu 或 Magisk
3. 连接手机：通过 USB 线连接电脑

## 常见问题

**Q: 找不到开发者选项？**
A: 设置 → 关于手机 → 连续点击版本号 7 次

**Q: 如何获取 Root 权限？**
A: 解锁 Bootloader → 刷入 KernelSu 或 Magisk

**Q: 清理后手机变慢？**
A: 正常现象，应用会重新生成缓存，使用几次后恢复正常

## 安全提示

- 清理前建议备份重要数据
- 微信/QQ 建议在应用内清理
- 备份镜像建议保留
