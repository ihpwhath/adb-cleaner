# ADB Cleaner v2.0 - Android 手机深度清理助手

Android 手机深度清理工具，通过 ADB + Root 权限扫描并清理垃圾文件。

---

## 一键安装

在 OpenClaw 中说：

`
安装这个 skill：https://github.com/ihpwhath/adb-cleaner
`

---

## 快速开始

### Windows 用户（最简单）

1. 下载 db-cleaner-v2.0.zip
2. 解压后双击 quick-clean.bat
3. 按 Y 确认清理

### OpenClaw 用户

安装后说：帮我清理手机

---

## 前置要求

- USB 调试已开启
- Root 权限已获取
- 手机已连接电脑

---

## 功能特性

- 自动检测设备和权限
- 智能分类垃圾文件
- 一键清理，安全可靠
- 生成详细清理报告
- 支持 Windows / OpenClaw

---

## 清理内容

**可自动清理：**
- 应用缓存（1-3 GB）
- 系统日志（50-100 MB）
- 应用日志（100-300 MB）
- 下载残留（100-500 MB）
- 临时文件、空目录

**建议保留：**
- 备份镜像
- 通话录音
- 用户数据
